三星發布新智慧手錶介面One UI Watch，強化與安卓手機間連結

三星放棄自家平台Tizen的主要原因，是希望透過強強聯手，與佔有智慧手錶市占40%的Apple Watch抗衡。在上個月的Google I/O 2021開發者大會中，Google就已宣布將與三星聯手開發新的智慧手錶介面。

Samsung於 MWC 2021 6/29 發表智慧手錶新平台 One UI Watch，三星表示此介面旨在優化 Galaxy Watch 智慧手錶與android手機體驗的連結，Galaxy Watch 專用的 One UI Watch，可透過與 Google 共同打造的平台安裝下載，以提升裝置的運作效能、強化 Galaxy Watch 與 Android 手機間的無縫連結，並提供更多應用程式體驗。

Galaxy Watch上的全新平台，可透過Play商店下載YouTube Music、Google Maps等Google原生的應用程式，以及各種熱門的第三方的應用程式。
