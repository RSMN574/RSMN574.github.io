Windows 11有望加入 Android 模擬器

早前有消息透露，Microsoft 將會在下一個版本 Windows 之中，加入 Android 模擬器功能。
Android 模擬器在 Windows 上執行有不少方式，而最近有人發現 Microsoft 官方用於執行 Linux GUI 應用程式的 WSLg 工具更新中，提及到新版本修復了「Android 模擬器」的一些問題，意味着這個功能很可能實際存在。


根據之前消息，微軟內部稱為 Project Latte 的計劃將容許開發者將 Android App 包裝成 MSIX 在微軟程式商店發行，並在 Windows 透過 Android 模擬器執行，這樣會更無縫，亦容許 Android 開發者接觸更多用戶。
